# Code of Conduct

## Standards

Contributors must communicate professionally, review technical claims against evidence, protect credentials and personal media, and respect applicable law and consent requirements.

Harassment, threats, doxxing, discriminatory conduct, malicious credential collection, non-consensual likeness workflows, and deliberate concealment of security defects are not accepted.

## Enforcement

Report conduct issues privately to the maintainers through the security contact configured for the repository. Maintainers may remove content, restrict participation, or reject contributions that violate these standards.
